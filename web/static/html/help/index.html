QA 常见问题

  • 忘记密码怎么办？

    登录密码遗忘：可通过资金密码在平台登陆口重新设定登录密码。

    资金密码遗忘：但有登录密码请联系在线客服协助处理。

    建议您初始拿到帐号请到平台“个人中心”下“用户详情”页面进行登陆密码、资金密码的设定；并请妥善保管好您的资金密码，如果资金密码和 登录密码遗忘将无法再修改和找回密码。

  • 如何绑定银行卡？

    在平台【个人中心】下点击“我的银行卡”，验证资金密码无误后即可进入银行卡信息页面，点击“绑定银行卡”，按照提示步骤即可绑定银行卡。

  • 无法绑定银行卡的原因：

    您的这张银行卡在之前是否被绑定过（不可以重复绑定）；

    您的开户人姓名是否与已绑定的银行卡开户人是一样的；

    点击“绑定银行卡”后的第一步是验证已经绑定在平台上的银行卡，这一步需要输入的是已绑定的银行卡信息，不是新绑定的银行卡信息。

  • 平台充值限额及服务时间？

    “工行线下充值”

    服务时间：每日上午09:00至次日凌晨02:00。

    充值限额：单笔最低充值50，最高充值40000。

    “农行线下充值”

    服务时间：每日上午09:00至次日凌晨02:00。

    充值限额：单笔最低充值100，最高充值40000。

    “建行线下充值”

    服务时间：每日上午09:00至次日凌晨02:00。

    充值限额：单笔最低充值100，最高充值40000。

    “招行跨行线下充值”

    服务时间：每日上午09:00至次日凌晨02:00。

    充值限额：单笔最低充值100，最高充值40000。

    “民生跨行线下充值”

    服务时间：每日上午09:00至次日凌晨02:00。

    充值限额：单笔最低充值100，最高充值40000。

  • 平台提现限额及服务时间？

    平台提现支持“工商银行”、“农业银行”、“建设银行”、“招商银行”、“民生银行”

    服务时间：每日上午10:00至次日凌晨02:00。

    提现限额：单笔最低提现100，最高提现50000。

  • 充值、提现如何免手续费？？

    同行充值返手续费：工行充值金额大于等于100；民生充值金额大于等于200；招行同行充值金额大于等于800；

    跨行充值返手续费：工、农、建、交转民生，充值大于等于100；招行转民生，充值大于等于1000；农行、建行转招行，充值大于等于800；

    提现：单笔提现金额100及100以上，并且投注额达到2天累计充值金额的10%后，提现免手续费。

  • 为什么充值没有即时到帐？如果出现以下问题，请及时联系在线客服并提供银行回单截图。

    汇款时未使用平台提供的最新银行信息；

    汇款时附言（充值编码）填写错误或者未填写；

    平台填写金额与实际汇款金额不一致；

    充值时间在平台服务时间外。

  • 游戏撤单问题？

    任何彩种游戏的撤单都必须在停止投注之前进行，当期投注时间截止后将无法再进行撤单操作；

    追号单的撤单需要到“追号任务信息”里面进行终止任务。

    超过5000的撤单，将收取0.5%的手续费

  • 什么是“奖金限额”？平台奖金限额是多少？

    每个账号在同一游戏、同一玩法、同一奖期中购买相同号码的最大可中奖金额。

    高频彩（时时彩、时时乐、11选5）的奖金限额40万，低频彩（3D、排列三/五）的奖金限额10万。

  • 关于第3方开奖机构不开奖问题？

    所有游戏，如遇第3方开奖机构不开奖,如果平台当日销售结束后官方依然没有开出号码,平台会对未获得开奖号码的期数进行撤单返款。撤单后无论第三方以任何形式补开，平台均维持撤单处理.
